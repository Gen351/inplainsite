About InPlainSite

InPlainSite is a decentralized, local-first utility designed to securely obfuscate and share private ledger credentials and keys across the web using standard public databases.

How it works:

  • Browse: Look through the catalog feed of decentralized records.
  • Decrypt: Supply both cryptographic parameters (Key A and Key B) to run absolute offline AES-256 local decryption in your sandbox.
  • Playground: Standard sandbox for dual-key encrypting any block of data locally.
  • Publish: Securely authorize and push new obfuscated objects to the public ledger list.
